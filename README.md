# 3.thermal-system-prediction-ml


## 📌 Project Description
This project uses Machine Learning (Linear Regression) to predict missing ambient temperature values based on system warm-up time.

## ⚙️ Workflow
- Data creation and preprocessing
- Handling missing values
- Training a regression model
- Predicting unknown temperature values

## 🧠 Technologies
- Python
- Pandas
- Scikit-Learn

## 🚀 Goal
To demonstrate a simple real-world regression workflow for sensor-based thermal data analysis.

## 📊 Output
The model fills missing temperature values using learned patterns from data.
